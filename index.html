About This Project

This project demonstrates the design and implementation of a modern school network using a two-tier architecture. The network is built with redundancy, scalability, and security in mind.

It includes dual routers configured with HSRP for gateway redundancy, Layer 3 core switches for inter-VLAN routing, and access switches for end devices in different departments.

Network Technologies Used

  • VLAN segmentation for departmental isolation
  • HSRP for high availability of default gateway
  • OSPF for dynamic routing across the network
  • DHCP and DNS services for automatic configuration
  • NAT for secure internet access
  • EtherChannel for increased bandwidth and redundancy
